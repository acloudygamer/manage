## Markdown 基础

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本格式

```markdown
**加粗**
*斜体*
~~删除线~~
```

### 列表

```markdown
- 无序列表
1. 有序列表
```

### 代码

```markdown
`单行代码`

```
多行代码
```
```

### 链接

```markdown
[文字](URL)
```

### 预览

VSCode 中按 `Ctrl + Shift + V` 预览。
